# Как сменить шаблон резюме

## Доступные шаблоны (8 штук)

| Шаблон | Подходит для |
|--------|-------------|
| Минималистичный | IT, стартапы, дизайн |
| Классический | Юриспруденция, финансы, госслужба |
| Современный | Маркетинг, PR, медиа |
| Двухколоночный | Опытные специалисты с большим объёмом |
| ATS-оптимизированный | Крупные корпорации, hh.ru массовый поиск |
| Международный | Работа за рубежом, LinkedIn |
| Академический | Наука, образование, исследования |
| Творческий | Дизайнеры, UX, креативные профессии |

## Как сменить шаблон

**В боте:**
1. Откройте последнее резюме командой /resume
2. Нажмите **«Сменить шаблон»**
3. Выберите из списка → PDF пересоздастся автоматически

**На сайте:**
1. Раздел «Резюме» → «Мои резюме»
2. Кнопка **«Шаблон»** → выберите → «Применить»

## Совет
Для hh.ru и SuperJob рекомендуется шаблон **«ATS-оптимизированный»** — он обходит автоматические фильтры.
