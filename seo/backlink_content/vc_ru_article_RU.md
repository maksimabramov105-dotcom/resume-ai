# vc.ru — Статья о стартапе

**Заголовок:** Мы автоматизировали поиск работы: 300 откликов в неделю, 12 собеседований, 3 оффера

**Раздел:** Стартапы, Карьера, ИИ

---

Идея родилась из личной боли: после сокращения пришлось тратить по 3-4 часа в день на рутину — редактировать резюме, писать сопроводительные письма, заполнять формы. При том что работа почти везде одна и та же.

Мы решили это автоматизировать. Так появился РезюмеАИ.

## Что делает продукт

**РезюмеАИ** — это Telegram бот с двумя режимами:

**1. AI Конструктор Резюме**
Загружаете базовое резюме, вставляете ссылку на вакансию — через 30 секунд получаете PDF-резюме, адаптированное именно под эту должность. GPT-4 переписывает формулировки, расставляет нужные ключевые слова, выделяет релевантный опыт.

**2. АвтоОтклик**
Настраиваете фильтры (должность, зарплата, город, ключевые слова, чёрный список компаний) — бот сам ищет подходящие вакансии на hh.ru, SuperJob, LinkedIn и отправляет отклики с персонализированным резюме для каждой позиции.

## Результаты первых пользователей

Средняя статистика по пользователям на платном тарифе за первый месяц:
- 200-300 откликов за 7 дней
- 10-15% callback rate (против 2-3% при стандартной рассылке)
- Среднее время до первого собеседования: 4 дня

## Бизнес-модель

**Бесплатно:** 3 автооткликов/день + 2 AI-резюме/день

**Базовый** (499 руб/мес): 50 откликов/день, все источники вакансий

**Профессиональный** (999 руб/мес): безлимит, LinkedIn автоматизация, приоритетная обработка

**Карьерный** (2499 руб/мес): личный карьерный ассистент + анализ рынка + переговоры по зарплате (AI)

## Технический стек

Python + aiogram 3 + FastAPI + OpenAI API + Playwright + hh.ru API. Деплой на одном VPS, мониторинг через собственный health check скрипт с алертами в Telegram.

Стоимость OpenAI на 500 генераций резюме в месяц: ~$10. Маржинальность высокая.

## Планы

- Интеграция с LinkedIn Jobs API
- Анализ рынка: 'сколько платят за вашу должность в вашем городе'
- Трекер отказов с аналитикой — почему не берут
- B2B: рекрутинговые агентства как клиенты
