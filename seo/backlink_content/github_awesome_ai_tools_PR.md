## Tool details

**ResumeAI** is an AI-powered Telegram bot that automates the job search process end-to-end.

**Core AI features:**
- GPT-4 based resume tailoring — reads the job description, extracts requirements, and writes a targeted resume in the candidate's voice
- PDF generation with clean formatting (reportlab)
- Cover letter generation (optional)
- Interview prep Q&A based on the specific role

**Automation features:**
- Scrapes hh.ru and SuperJob via official APIs
- Applies to matching vacancies automatically
- LinkedIn Easy Apply automation via Playwright
- Configurable filters: salary range, keywords, blacklist companies, experience level

**Tech stack:** Python 3.11, aiogram 3, FastAPI, OpenAI API, Playwright, SQLite, systemd
